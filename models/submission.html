Submission¶

Contains the basic information of a project submitted by one team.

class Submission¶
id¶

int (primary key)

Unique ID of the Submission

title¶

string

Title of the project. Try to be creative!

description¶

string

Description of the project. May be used to show the basic idea behind the project

url¶

string

Url to the project source code. This may be a repository or a meta site that includes a presentation of the project, or instructions, in addition to the source code

team_id¶

int (foreign key to Team)

ID of the team that owns this submission
