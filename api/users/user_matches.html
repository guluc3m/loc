/users/<user>/matches¶

GET /users/(string: username)/matches¶

Show matches in which the user has participated.

Parameters:
  • username (string) – the username for which to obtain information
Response JSON Object:
 
  • id (int) – unique id of the match
  • title (string) – title of the match
Status Codes:
  • 200 OK – when user record was found. Returns user information
  • 404 Not Found – when user record was not found or was deleted

Example request:

GET /users/testuser/matches HTTP/1.1
Host: example.com
Accept: application/json
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/json

[
    {
        "id": 42,
        "title": "Practice match",
    }
]
