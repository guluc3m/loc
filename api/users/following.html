GET /profile/following¶

Obtain current logged in user is following.

Request JSON Object:
 
  • token (string) – JWT token of the active user session
Response JSON Object:
 
  • username (string) – username followed
Status Codes:
  • 200 OK – when user is logged in; returns list of followed users
  • 401 Unauthorized – when the user is not logged in
  • 404 Not Found – when user record cannot be found or has been removed

Example request:

GET /profile/following HTTP/1.1
Host: example.com
Accept: application/json
Content-Type: application/json

{
    "token": "JWT_TOKEN"
}

Example response:

HTTP/1.1 200 OK
Content-Type: application/json

[
    "frienda",
    "friendb"
]
