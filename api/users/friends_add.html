/profile/friends/add¶

POST /profile/friends/add¶

Add a new friend to the user’s profile.

This should create a friend request and notify the added user about it so that they can accept/reject the invitation.

Friend associations work 2-way.

Request JSON Object:
 
  • token (string) – JWT token of the active user session
  • username (string) – the username of the friend to add
Status Codes:
  • 200 OK – when user is logged in
  • 401 Unauthorized – when the user is not logged in
  • 404 Not Found – when user record has been removed or does not exist

Example request:

GET /profile/friends HTTP/1.1
Host: example.com
Accept: application/json
Content-Type: application/json

{
    "token": "JWT_TOKEN",
    "username": "frienda"
}

Example response:

HTTP/1.1 200 OK
Content-Type: application/json
